Introduction of ESP Board Overview

This page provides an overview of the ESP boards provided by the ESP cores (esp8266 and esp32) used by the Arduino IDE and Arduino CLI.
The board overview pages lists:
  • The board name
  • The board variant
  • The built in LED GPIO (if available)
  • The microcontroller (mcu) used by the board
  • The flash size of the board
  • A link to the pins_arduino.h file for the board variant
and provides filter and sort functionality.
The overview is generated from the boards.txt file and pins_aruino.h of the respective core and board variant.
If you are interrested how the data of overview are generated, you can have a look at the repository ESP_Board_Overview. This repository contains scripts and a docker dev container to generate this Angular website.
